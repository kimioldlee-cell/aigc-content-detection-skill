# AIGC 文章内容检测技能

一个系统化的 AIGC 文章检测技能，用于识别文章是否涉及 AIGC 标注规则中的各管控场景。

## 功能

- 六层分级检测：从硬证据到 AI 身份意图，逐层递进
- 文体识别：自动判断文体类型，不同文体使用差异化判定标准
- 高发误判区鉴别测试：六大误判场景的专门鉴别逻辑，防误杀真人创作
- 回归测试集：13 个 AI 正样本 + 8 个真人反样本，确保判定边界稳定

## 检测维度

1. **第一层：AIGC 硬证据** — AI 对话残留、创作声明、工具截图、生成配图、故障残留
2. **第二层：语言与文风** — 词汇空洞、句子节奏规范化、活人味缺失
3. **第二层：结构与排版** — 排版工整、结构套路、逻辑衔接异常
4. **第三层：内容与事实** — 内容全面不深入、强行升华、捏造不实
5. **第三层：真人创作反证** — 真实经历、个人立场、专业见解、运营痕迹
6. **独立维度** — AI 味主观感知、AIGC 生产推广内容、AI 身份意图

## 安装

将此 SKILL.md 放置于 Hana Agent 的 skills 目录中。

## 使用

```
帮我检测这篇文章是否是AIGC生成的：[文章内容]
```
