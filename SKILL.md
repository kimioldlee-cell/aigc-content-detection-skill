# AIGC 内容检测技能

## 功能
识别文章是否涉及 AIGC 标注规则中的各管控场景（特征项），对每个特征项输出标记：1=涉及，0=未涉及。

## 使用方式
用户提供一篇文章（文本、链接、或文件），技能逐项检测并输出结果。

## 检测维度与特征项

### 第一层：AIGC 硬证据审核（命中任一即判定"AIGC-硬证据"）

| 特征项 | 判定规则 |
|--------|----------|
| AI对话残留 | 1. prompt/指令残留：正文出现作者发给AI的指令或提示词（如"请帮我写一篇关于XX的文章，要求字数800字，风格活泼"）；2. AI套话开头：以AI典型口头禅开头或收尾（如"好的""以下是……""作为AI语言模型，我……""当然可以，让我来为您……"）；3. 对话格式残留：保留了AI的一问一答对话结构（如"问：……答：……"），或直接把聊天框多轮对话粘进文章；4. AI第一人称口吻：以AI视角、第一人称撰写（如"主人问我"） |
| AI创作声明 | 文末/文中直接标注或声明"AI创作"/"AI生成"/"由xxx AI撰写"/"以下是xxAI生成的回答"等 |
| AI工具截图 | 文内配图有AI工具界面截图，且工具生成截图显示文章本身由AI生成，同时文章非AI教程等科技垂类文章 |
| AI生成配图 | 文章内配图有明确AI生成痕迹，如AI美女、AI创作的风景、六指、文字乱码、AI水印等 |
| AI生成故障残留 | 正文出现以下任一机器生成故障痕迹：1. **机械复读**：整段或整句内容近乎逐字重复出现多次（非刻意的修辞重复，而是同一信息块被原样拼贴2次以上）；2. **占位符/指令泄露**：未清理的模板占位符（如"[此处插入XX]""{{topic}}"）、或半删除的系统指令残留（如"作为一个AI语言模型"被部分删掉只剩"作为一个…模型"）；3. **半截断裂**：文章在某处突然中断、句子只写了一半、或出现明显的拼接接缝（前半段和后半段主语/时态/风格突变，像两次生成硬拼在一起） |

### 第二层：语言与文风

| 特征项 | 判定规则 |
|--------|----------|
| 词汇宏大空洞 | 1. 滥用修辞：偏好用宏大、高级、晦涩的词汇或概念来做比喻，辞藻宏大但形式大于内容；2. 套话堆砌：高频使用高级词汇，看似专业实则毫无信息量（如赋能、助力、抓手、闭环、底层逻辑等）；3. 字数灌水：一句话能说明白的事，AI会用不同的从句和同义词翻来覆去说三遍 |
| 句子细节极度规范化 | **核心是"句子节奏缺乏人的呼吸感"，不是单纯的形态均匀。** 1. 节奏缺乏起伏：句子全程同温、匀速输出，该急的地方不急、该停的地方不停，缺乏长短句交错的"突发变化"，读着四平八稳——注意这指的是"节奏没有人的呼吸感"，而非简单的"句子长度接近"；2. 标点符号极度规范：中文章里破折号（——）密度异常偏高，频繁地用于补充说明；中英文混用的时候会全部隔一个半角空格，细节标准到反常。<br>**注意：句长均匀/一句一段既不是必要条件、也不是充分条件**——见下方"误判高发区二"的鉴别要求 |
| 缺乏活人味 | 1. 语气机械，没有个人态度：全程平稳客观，没有惊讶、吐槽、激动、无奈等真实情绪；2. 人机感重：写体验/感受像在企广告词，空有华丽辞藻却没有真实细节；3. 没有道德瑕疵：很少表现出自私、阴暗、偏激、小肚鸡肠等真实人性 |

### 第二层：结构和排版

| 特征项 | 判定规则 |
|--------|----------|
| 排版极度工整 | 1. 机械分点：严格用"一、二、三"、"一是、二是、三是"、"●要点"的分点形式；2. 排版形式过度：滥用markdown格式排版，每段首句加粗后跟冒号解释，不重要的地方也滥用粗体 |
| 行文结构套路化 | 1. 文章整体结构套用标准框架，骨架千篇一律（如套用"首先-其次-最后"/"总-分-总"/"问题-影响-解决方案"/"背景→定性→建议"等）；2. 段落之间结构高度同质：各段采用统一展开模式，每段长度/结构都差不多；3. AI常见句式反复出现：通篇反复出现"不是...而是..."否定结构、三级排比句、对称句等句式 |
| 逻辑衔接异常 | 1. 连接词硬堆，推导缺失：句子之间靠"因此""所以""由此可见"等连接词强行拼在一起，但实际上前后的因果推导是断的；2. 话题乱跳，内容重叠：主题经常莫名其妙地跳——比如上一段还在讲技术原理，下一段突然扯到社会治理 |

### 第三层：内容和事实

| 特征项 | 判定规则 |
|--------|----------|
| 内容全面但不深入 | AI喜欢把一个话题从背景、意义、优势、劣势、未来等各个角度都铺一遍，看着很"全面"，但每个点都只说一两句，浅尝辄止，通篇是正确的"常识汇总"，没有新鲜、独到、带个人思考的深刻观点 |
| 强行升华 | 1. 强行给正能量结局：前面讲的事有多么黑暗、绝望、悲惨，文章的最后一段一定会像新闻联播一样给出正向的结局；2. 小事上价值：AI总是把一件微不足道的小事拔高到人类、时代、社会这样的高度；3. 争议话题和稀泥：在讲某个有争议的负面社会新闻时，文末结尾和语调一转告诉读者"我们要保持理性和宽容" |
| 捏造不实 | 1. 编造模糊案例：AI喜欢捏造没有时间、地点、人名的模糊案例，用以拉近读者距离（如"上周小李在公司发生了这样一件事……"）；2. 伪造权威来源：编造不存在的专家名字、不存在的学术期刊/文献，甚至是假冒的法律条文等；3. **精装假案例**：编造一个"具体时间+真名+完整流程"看起来很真实、但每个细节都服务于主题、情绪得体向善的案例（详见下方"真实经历 vs AI精装假案例"鉴别） |

### 第三层：真人创作反证（出现以下特征说明文章有真人创作痕迹）

| 特征项 | 判定规则 |
|--------|----------|
| 真实个人经历 | 文中有真人亲历的质感。**注意：判定依据不是"有真名+精确时间"**（真人写生活记述出于隐私和习惯，反而常用"我同事""老王""上个月"这类模糊指代，很少写真名和精确时间）。真正的判定看是否有"只有亲历者才会有"的真实质感信号（详见下方"真实经历 vs AI精装假案例"鉴别） |
| 明确个人立场和情绪表达 | 文章有明确的个人立场和真实情绪，作者表达了清晰的主观态度——可能是愤激的、带情绪的、甚至有争议的，而不是"一方面另一方面"的端水话术 |
| 专业独到见解 | 文章里有该领域的深度见解、行业内幕、非公开的一手信息——不是网上搜得到的常识汇总，而是"只有真正干这行的人才知道"的内容 |
| 真人运营痕迹 | 账号有长期真人运营痕迹（真实评论互动、线下活动关联、社群联动等） |

> - **弱AI感**：有一些AI味，但不确定

> 从内容治理角度，这类文章是AI水文的"生产源头和方法论传播"，往往比单篇AI文更值得关注。

> 2. 传授用AI批量产文薅流量主/带货收益的完整工业流程（养号→对标抄选题→AI生成→定时发布）
> 3. **教如何绕过平台AI检测**（如"用朱雀/易撰检测AI率，人工特征要达80%""把生硬的地方改成自己的话降AI率"）——这是最强信号
> 4. 提供选题库/对标账号库/造号话术，目标是批量复制账号
>
> ⚠️ 注意区分：正常的"AI工具测评/教程"（科技垂类，客观介绍工具功能）**不命中**；只有以"批量造号薅流量/洗稿/对抗检测"为导向的才命中。

> 这是与"本文是否AI生成"独立的维度，用于区分AI内容的**创作意图**，仅在前几层已判定为AI生成时才需要标注。从内容治理角度，"试图冒充真人"的AI内容危害更大（误导读者把AI内容当真人观点/真实经历），处置应更严。
>
> - **冒充真人**：AI生成但伪装成真人写作（编造个人经历、假客户案例、第一人称真人口吻），意图让读者误以为是真人。**绝大多数AI水文属此类**（鸡汤文、伪专业案例文等）

> - **未表态**：纯功能性/资讯性内容，无明显身份伪装意图（如AI生成的攻略、科普）

## 文体识别（前置步骤）

在逐项检测之前，先判断文章所属文体类型：

| 文体类型 | 示例 |
|---------|------|
| 文学创作 | 小说、散文、诗歌、剧本、同人文、网文连载 |
| 观点/议论 | 时评、社论、观点文、公众号评论 |
| 知识/科普 | 教程、科普文、百科类、技术文章 |
| 营销/商业 | 产品软文、品牌故事、广告文案 |
| 纪实/新闻 | 新闻报道、人物特稿、纪实文学 |
| 个人表达 | 日记、随笔、朋友圈、个人博客 |

## 文学创作类文体的特殊判定规则

当文章被识别为**文学创作**类（小说/散文/诗歌/剧本等）时，以下特征项的判定标准需要调整：

### 不直接命中的情形（文学创作中的正常手法）

| 特征项 | 文学创作中"不算违规"的情形 | 仍然算违规的情形 |
|--------|--------------------------|----------------|
| 词汇宏大空洞 | 文学性比喻、意象营造、诗化语言（有具体意象承载） | 空洞套话堆砌，比喻没有具体画面，纯粹辞藻轰炸无叙事功能 |
| 句子细节极度规范化 | 刻意的排比用于节奏/情绪渲染（如余华、莫言的风格化写法） | 全文所有段落句长均匀、无任何节奏变化，排比不服务于情节而是填充字数 |
| 缺乏活人味 | 冷叙事风格（如加缪《局外人》式克制）是文学选择 | 所有角色情绪反应用同一模板（"沉默了几秒""眼睛红了但没流泪"），感受不到作者对角色的独特理解 |
| 排版极度工整 | 章节分明、段落有节奏是小说正常结构 | 每段首句加粗+冒号解释、滥用markdown格式（小说中不应出现） |
| 行文结构套路化 | 线性叙事、三幕结构、起承转合是经典叙事框架 | 每个场景内部用完全相同的"环境描写→外貌→对话→抒情独白"模板展开，无变化 |
| 强行升华 | 文学作品结尾的主题升华、象征收束是正常手法 | 升华与前文情节脱节，像是另外贴上去的"金句"；或每隔几段就升华一次，密度不正常 |

### 文学创作类的核心判断标准

对于文学创作，判定AI生成的关键不在于"用了什么手法"，而在于：

1. **手法是否服务于叙事/情感**：真人作者的修辞、排比、象征都有叙事功能、推动情节或塑造人物；AI的修辞是装饰性的，删掉不影响叙事
2. **节奏是否有呼吸感**：真人作者有张有弛，AI全程匀速
3. **角色是否有独立声音**：真人笔下角色有可辨识的性格差异和说话方式；AI笔下角色共享同一种思考方式
4. **细节是否有"冗余的真实"**：真人会有对叙事无直接用处但很具体的旁枝末节（人就是这样记忆的）；AI的细节全部服务于情节

## 多故事文章（故事集/鸡汤合集）的AI判别逻辑

### 第一步：识别故事集结构
文章由多个独立小故事串联组成（常见为3个故事+道理提炼），先判定为故事集文体。

### 第二步：故事集专属检测维度

| 检测维度 | AI故事集的特征（命中=AI倾向） | 真人故事集的特征（命中=真人倾向） |
|---------|------------------------------|--------------------------------|
| 案例来源 | "后台收到一个女孩的信息""我有个朋友/发小""听说有个人"——来源模糊、不可追溯、像随时可替换的素材 | 有具体可追溯的来源：明确的时间地点人名、作者亲历、可考证的公开事件/新闻 |
| 细节质感 | 细节是"功能性"的，只为佐证论点而存在，删掉故事不影响道理（如杯子、围巾、小米粥都是标准道具） | 细节有"冗余的真实感"——存在与论点无关但很具体的旁枝末节，因为真事就是这样发生的 |
| 故事间结构 | 三个故事结构高度同构：篇幅相近 → 都是"场景+一个细节+点题"，像一个模板套三次 | 几个故事详略不一、结构各异，有的长有的短，节奏不规整 |
| 道理提炼 | 每个故事后都接金句/名人名言/影视引用，升华密度均匀且套路化 | 提炼方式多样，有的直接说有的留白，不是每段都强行给金句 |
| 主角"声音" | 不同故事的主角共享同一种说话/思维方式，看不出人物差异 | 不同主角有可辨识的差异（用词、性格、处境的具体性不同） |

### 第三步：故事集的判定规则

1. **每个故事单独过"真实经历 vs AI精装假案例"测试**（见下方误判高发区四）。
2. **统计假案例占比**：
   - 多数故事（≥半数）为模糊不可追溯的精装假案例 → 直接强命中「捏造不实」，且嫌疑等级 ≥ 高
   - 故事大多有可追溯真实来源 → 不命中「捏造不实」，按常规逻辑走
3. **结构同质化加权**：若多个故事结构高度同构（同模板套用），叠加命中「行文结构套路化」。
4. **真人反证的特例**：故事集中只要有**1 个**故事是经得起推敲的作者亲历（具体时间地点人物、有无关旁枝），即可计入「真实个人经历」反证；纯转述他人/后台投稿的故事不计入。
5. ⚠️ **不要因为"是故事集"就豁免**：真人写的故事合集（如真实采访稿、亲历记录合集）能通过真实性检测；AI鸡汤文虽然也是故事集，但故事全是模糊假案例，会在真实性检测上集体失分——区别就在这里。

## 易误判项的鉴别测试（输出前必做）

某些特征项容易把"真人的好表达"误判为"AI特征"。命中以下任一项之前，必须先做对应的鉴别测试，**通过测试才能确认命中**。

### 误判高发区一：文学性短句 vs 词汇宏大空洞

**问题**：文学性的凝练短句（如"星星被车轧过，还是星星"）和AI的词汇膨胀（如"开启那开会的进程"），表面都"不说人话"，但本质相反——前者是**信息压缩**（删不得），后者是**信息膨胀**（删得掉）。

判定"词汇宏大空洞"前，对可疑句子逐一做三个测试：

1. **白话还原测试**：能否用更短的大白话完全等价替换，且不丢失任何意味？
   - 能完全替换（"开启那开会的进程" = "开会"）→ AI空洞，**命中**
   - 不能（替换后韵味/张力/言外之意丢失）→ 文学表达，**不命中**

2. **删字测试**：删掉修饰词/从句后，核心信息是否丢失？
   - 不丢（说明这些字是填充物）→ AI膨胀，**命中**
   - 丢失（每个字都承载意义）→ 文学凝练，**不命中**

3. **意图测试**：这个表达是在制造留白/张力/回味，还是单纯把短话拉长显得高级？
   - 拉长凑字 → **命中**
   - 制造文学效果 → **不命中**

**判定规则**：三个测试中，文学表达通常全部"不命中"；AI空洞通常全部"命中"。混合情况下，看全文该类句子的**主导倾向**——若多数是"删不得"的凝练，则不命中本项；若多数是"删得掉"的膨胀，则命中。

> 对照示例：
> - "开启那开会的进程"="开会"（白话可替换、删字不丢、纯拉长）→ AI空洞 ✓命中
> - "产生了置信之感"="自己也信了"（白话可替换）→ AI空洞 ✓命中
> - "星星被车轧过，还是星星"（无法用白话等价替换，删字即失韵味，制造回味）→ 文学凝练 ✗不命中
> - "他选择了炸弹，不是火炬"（比喻有张力，删改即失意味）→ 文学凝练 ✗不命中

### 误判高发区二：句子节奏 vs 句子细节极度规范化

**关键认知：句长均匀、一句一段、句子短，这些"形态特征"既不是命中的必要条件，也不是充分条件。**

- **不是必要条件**：AI规范化可以表现为长句也可以表现为短句，不一定句长接近才算。命中的真正依据是"标点异常规范"或"节奏缺乏人的呼吸感"，二者满足其一即可。
- **不是充分条件**：很多真人写作刻意采用"短句、一句一段、句长均匀"的形态，这是合理的风格选择，不能据此判命中。

**判定前必须先排除以下真人合理形态**（这些情况下"句长均匀/一句一段"是风格，不命中）：
- 新媒体/公众号排版（为手机端易读，故意一句一段、多分段）
- 诗歌、散文诗（短句分行本就是文体特征）
- 极简主义文风（如海明威式硬汉短句、留白美学）
- 段子手/营销文案（靠短句制造节奏感、停顿感、爆点）

**真正的命中依据：句子节奏缺乏"人的呼吸感"。** 用下面的对照来判断方向：

| 信号 | 倾向 |
|------|------|
| 短句之间有真实情绪起伏、有意外转折、有个人化的停顿与加速，该急时急该缓时缓 | 真人风格化写作 → **不命中** |
| 短句全程同温、无情绪、内容是套话、像匀速机器输出，每一句的"力度"都一样 | AI规范化 → **命中** |

一句话：真人的短句是**有呼吸的**（该停的地方停、该急的地方急），AI的短句是**匀速的**（每句力度一致、全程同温）。判命中的不是"句子均匀"这个形态，而是"均匀背后没有人的节奏"。

> 此外，如果文章同时出现"标点异常规范"（破折号密度异常偏高、中英文混排机械加半角空格），可独立作为本项命中的依据，不依赖句子节奏。

### 误判高发区三：文学升华 vs 强行升华

判定"强行升华"前，确认结尾升华是否与前文情节有机关联：
- 与前文意象/情节呼应，是叙事的自然落点 → 不命中
- 与前文脱节、像外贴的金句、或升华密度过高（每隔几段就升华）→ **命中**

### 误判高发区四：真实经历 vs AI精装假案例

**问题**：很多人误以为"有真名+精确时间=真实，模糊指代=编造"。实际相反——真人写生活记述出于隐私和习惯，常用"我同事""老王""上个月"这类模糊指代，很少写真名和精确时间；而"具体时间+真名+完整流程"的过度精确案例，反而常是AI在刻意伪造真实感。

**核心判据：真人记述靠"无用的真实噪音"取信，AI案例靠"有用的精装细节"伪装。**

判定"真实个人经历"（反证）或"捏造不实-精装假案例"前，检查以下5类真实质感信号：

| 信号 | 真人亲历（→真实个人经历命中） | AI编造（→捏造不实命中） |
|------|------------------------------|------------------------|
| **1. 无目的的冗余细节** | 有对主题毫无用处但就是记得的细节（"那天下着雨，我鞋进水了，袜子湿了一整天"） | 每个细节都服务于要讲的道理，没有无关的真实噪音 |
| **2. 感官与身体记忆** | 记得气味、温度、身体不适、当时吃什么等"在场感" | 只写事件流程，缺身体层面记忆 |
| **3. 情绪的不体面** | 承认当时"想骂人""有点幸灾乐祸""怂了没敢说"等不光彩反应 | 人物情绪得体、合理、最终向善 |
| **4. 模糊的方式** | 该模糊的模糊（老王、我表姐），但某个具体瞬间清楚得反常（"他手里还捏着半个包子"） | 全程均匀模糊，从头到尾没有一个钉死的、立得住的细节 |
| **5. 细节的不可推导性** | 细节很私人、很怪、很具体，"你编也编不出来" | 都是"能想到的标准细节" |

**判定规则**：
- 命中3+个"真人亲历"信号 → "真实个人经历"反证命中
- 出现"具体时间+真名+完整流程"但5类真实信号全无、且情绪得体向善、每个细节都有用 → "捏造不实-精装假案例"命中
- 模糊指代本身**中性**，不能因为"没写真名/没写精确时间"就否定真实性，也不能因为"写了真名/精确时间"就认定真实

> 对照示例：
> - "上周小李在公司发生了这样一件事，他认识到了……"（全程均匀模糊、细节都服务道理、无身体记忆）→ AI模糊编造 ✓捏造不实
> - "我妈住院那十几天，我天天在医院走廊的折叠椅上睡，椅子缝硌得我腰疼，护士站的灯一晚上不关，我记得隔壁床大爷打呼噜像拖拉机"（无用噪音+身体记忆+不可推导）→ 真人亲历 ✓真实个人经历

### 误判高发区五：生活化钩子开头 ≠ AI特征

**问题**：用一个看似与正文无关的生活场景开篇（"上周吃火锅时……""路过小区门口看到……"），是真人写作中极常见的引入手法，**绝对不能单独作为AI证据**。把"无关钩子"当成AI特征会大量误杀真人随笔、公众号、知乎体。

**核心：钩子开头本身中性，真人和AI都用。区别在钩子的"质感"和"是否回收"。**

| 维度 | 真人的钩子（→不命中） | AI的钩子（→可作辅助佐证） |
|------|---------------------|--------------------------|
| 细节质感 | 有真实、私人、不可替换的细节 | 空心的标准氛围道具（火锅/咖啡/夕阳，换掉无影响） |
| 与正文关系 | 后文有回扣、呼应或自然延展，钩子是文章有机部分 | 用完即丢，钩子和正文断裂，细节再不出现 |
| 落地锚点 | 钩子里的人/事能和全文其他真实信息形成锚点网 | 钩子人物与全文一样模糊不可追溯 |

**判定规则**：
- 仅凭"开头看似无关"→ **不命中任何特征项**
- 只有当「钩子细节空心 + 与正文断裂不回收 + 全文无任何真实锚点」**三者同时成立**时，才可作为「逻辑衔接异常」或「捏造不实」的**辅助佐证**，**不单独成项**
- 真人的生活化开场（哪怕和论点关系松散）一律放行

### 误判高发区六：伪专业案例文 / 单案例说理文（高难度，最易漏判）

**问题**：这是最狡猾的一类AI内容——用**真实存在的理论/方法论**（心理学、管理学、医学、盖洛普、MBTI、木桶理论等）做包装，配一个**完美服务论点的客户/学员案例**，再加引流转化。它靠"理论是真的""看起来很专业"骗过初判，是比鸡汤文高级得多的AI变体，大量出现在职场号、心理号、教练号、咨询号。

**典型结构**：痛点共鸣开场 → 抛出客户/学员案例（"我的客户小X""我的学员"）→ 制造认知反转 → 科普一个公开理论 → 给三步/五步方法 → 升华 → 引流转化（限时名额/优惠码/加微信）。

**两个关键鉴别点（必须分清）**：

| 容易误判为真人 | 实际判定 |
|---------------|---------|
| "引用了专业理论" → 以为是"专业独到见解" | ❌ **公开理论 ≠ 一手洞察**。盖洛普/木桶理论/MBTI 都是网上能搜到的常识。搬运公开理论套进案例，不构成「专业独到见解」反证。只有"只有真正干这行才知道、网上搜不到"的内幕，才算独到见解 |
| "有具体客户案例" → 以为是"真实个人经历" | ❌ **完美服务论点的案例 = 精装假案例**。用误判高发区四的5类真实信号检测：若案例人物模糊（只有代号）、细节全为论点服务、对话像定制台词、情绪得体向善、无身体记忆/无用噪音 → 命中「捏造不实」，而非真实经历 |

**判定规则**：
- 案例严丝合缝服务论点 + 人物不可追溯 + 无真实质感信号 → 命中「捏造不实」
- 全文专业内容都是公开可搜的理论复述 → 「专业独到见解」**不命中**（不给反证）
- 同时叠加结构套路化（咨询营销模板）、强行升华、引流转化召唤
- **真人运营痕迹可单独命中**（真实人设署名/微信/限额转化），但它只说明"账号是真人运营"，**不等于"正文是真人原创"**——常见情况是「真人教练 + AI代写/重度辅助」
- 综合：这类文章典型结论为「**嫌疑：中偏高 ~ 高**」，AI味「弱~强」，取决于案例编造痕迹的明显程度

> 对照：盖洛普"小Sun"案例文——理论（盖洛普34才干）是真的，但小Sun案例完美服务"别补短板用优势对冲"的论点、人物仅代号、对话像剧本 → 捏造不实命中、专业见解不命中 → 中偏高嫌疑。

## 功能性文档（教程/攻略/说明书）的AI判别要点

**问题**：操作教程、使用指南、攻略类内容本就该"步骤清晰、分点有序、配图引导"——这些是**文体要求**，不能当AI特征。但"功能性文档"**不能一律豁免判真人**，因为AI也能批量生成通用攻略。判别关键在于区分"不可替代的实操"和"可批量生成的信息聚合"。

**核心区分维度**：

| 维度 | 真人功能文档（→无嫌疑） | AI功能文档（→高嫌疑） |
|------|----------------------|---------------------|
| 真实锚点 | 绑定具体主体——某家医院、某个真实账号、真实界面截图、真实二维码，内容不可替代 | 全是通用流程，无任何独家/实拍/专属信息，换个主体照样成立 |
| 排版 | 朴素，跟着实拍截图走，不刻意 | emoji 工业级分级排版（🔍📱🔒+1️⃣2️⃣3️⃣嵌套+✅🚫清单），过度结构化 |
| 数据 | 不堆砌效果数据，或数据有出处 | "成功率90%""通过率提升40%""10分钟出结果"等无出处的伪精确数字 |
| 语气 | 机构/作者朴素引导 | 刻意的"贴心AI助手"口吻（"别慌""喝杯奶茶等着""自己的号自己救💪"） |
| 内容来源 | 本主体专属、需亲历才能制作 | 网上信息聚合，AI 可一键生成 |
| 升华 | 教程就是教程，不升华 | 把操作教程强行升华成励志金句收尾 |

**判定规则**：
- 有真实专属锚点（具体机构/账号/实拍图）+ 排版朴素 + 无伪数据 → 判真人/无嫌疑
- 通用流程聚合 + emoji工业排版 + 伪精确数据 + 助手口吻 + 强行升华 → 判AI/高嫌疑
- ⚠️ 不要因为"是教程/有步骤"就直接豁免，必须查真实锚点

> 对照：医院绑卡教程（具体医院+实拍界面+真二维码+朴素排版）→ 真人；游戏解冻攻略（通用流程+emoji工业排版+"成功率90%"伪数据+"自己的号自己救💪"）→ AI。

## 输出前自查清单（强制）

在输出最终结果之前，必须完成以下自查，不允许跳过：

1. **文体确认**：是否已正确识别文体？文学创作类是否套用了特殊判定规则？
2. **易误判项复核**：对"词汇宏大空洞""句子细节极度规范化""强行升华""真实个人经历""捏造不实"等项，是否做了对应的鉴别测试？是否把真人的好表达误判为AI特征，或把模糊指代误判为"不真实"，或把AI精装假案例误判为"真实"？
3. **逐项反问**：每个标记为1的项，问自己"如果是个有水平的真人作者，会不会也这样写？"——若答案是"会"，重新审视该项判定
4. **整体一致性**：各项标记与综合嫌疑等级、AI味等级是否自洽？文学创作的容忍度是否已应用？
5. **证据具体性**：每个命中项是否能引用原文具体句子作为证据？无法举证的命中项应撤销

只有通过以上自查，才输出最终结果。

## 判定逻辑

### 步骤一：文体识别
先确定文章文体类型。

### 步骤二：逐项检测（含鉴别测试）
按各层特征项逐一检测。文学创作类参考特殊规则；易误判项必须先做鉴别测试。

### 步骤三：输出前自查
完成上方"输出前自查清单"，撤销无法举证或属于误判的命中项。

### 步骤四：综合判定

1. **第一层硬证据**：任一项命中即直接判定"AIGC-硬证据"，嫌疑等级=高
2. **第二层+第三层**（排除反证）：
   - 第二层命中 4+ 项 且 第三层内容事实命中 1+ 项 → 嫌疑等级=高
   - 第二层命中 3+ 项 → 嫌疑等级=中
   - 第二层命中 1-2 项 → 嫌疑等级=低
   - 第二层命中 0 项 → 无明显嫌疑
3. **真人反证降级**：如果"真人创作反证"命中 2+ 项，嫌疑等级下调一级（高→中，中→低，低→无）
4. **文学创作容忍度**：文学创作类文章，第二层命中阈值上调1项（即需要多命中1项才升级嫌疑等级）

## 使用示例

用户："帮我检测这篇文章是否是AIGC生成的：[文章内容]"

技能将逐项分析并输出上述格式的检测结果。

## 回归测试集（实测校准样本）

以下为已人工校准的样本，用于校验判别边界、防止改动后回归出错。修改技能逻辑后，应确保这些样本的判定结论不变。

### AI 正样本（应判：高嫌疑 / 强AI感）

| # | 文体 | 关键露馅点 |
|---|------|-----------|
| P1 | 长篇小说（悬疑庭审） | 角色声音同质化、情绪节奏匀速、"不是A是B"句式滥用、象征物反复翻译 → 文学容忍后仍判低~中，AI味弱~强（文学创作不享高判） |
| P2 | 情感号鸡汤（3故事集） | 三故事全为模糊不可追溯精装假案例、同模板套用、引用名人装权威、免责声明尾巴 |
| P3 | 励志短文（纯口号） | 全程空洞高级词+对称句、零具体内容、段段升华 |
| P4 | 历史科普（田氏代齐） | **硬证据：配图带"豆包AI生成"水印** → 直接锁高；史实虽准确但不影响判定 |
| P5 | 星座清单（三星座） | 并列列举集、每段同构、空洞标签、可替换性测试全失分 |
| P6 | 保险营销（9点列举） | 并列列举集九连发、对称句密集、空对空说理、段段升华 |

### 测试集核心校准结论（修改技能时务必保持）

1. **史实准确 ≠ 非AI**（P4）：AI可生成史实无误但带强AI味的内容，靠硬证据/文风判定
2. **专业理论 ≠ 独到见解**（P8）：搬运公开理论不给「专业独到见解」反证
3. **有案例 ≠ 真实经历**（P8/P9）：完美服务论点的案例是精装假案例
4. **功能性文档不一律豁免**（P10 vs N5）：查真实专属锚点区分真人/AI
5. **新闻客观 ≠ 缺乏活人味**（N3）：客观是新闻文体要求；精确真名+时间是真实性证据而非编造
6. **推荐用AI ≠ 本文AI写**（N6）：看正文内容质感，不看是否出现"AI"字眼
7. **生活化无关开头 ≠ AI**（误判高发区五）：钩子单独中性，不单独成项
8. **文学手法（排比/比喻/升华）在文学创作中不直接命中**：享受容忍度，靠角色同质化/节奏匀速等才暴露
9. **AI人格化内容区分"故意 vs 露馅"**（P11）：硬证据命中=AI生成（判定不变）；但需进一步区分「无意识AI生成（想冒充真人没装好）」与「有意识AI人格化创作（公开以AI身份写作、当卖点）」——两者均判AI，但内容治理处置逻辑不同，按需在「AI身份意图」附加标记中注明
10. **伪权威背书是辅助信号非硬证据**（P12）：泛泛搬出"心理学上说""专家表示""研究证明"却无具体出处/文献名，是AI增强可信度的惯用手法，计入「内容全面但不深入」或佐证整体AI味，但单独不强命中「捏造不实」（除非编造了具体的人名/期刊/法条）
11. **AI生成未精修的故障痕迹**（P12）：编号体系混乱、低级文字错别字（同音字/重复字）夹在高度工整句式中、机械复读——这类"工整与粗糙并存"的矛盾本身就是"AI生成+人工未精修"的强信号

12. **专业信息质感是真人 vs AI 的分水岭**（N7/N8 vs P8/P13——最重要的反误杀原则）：不能因为"写得专业、信息密集、有理论/参数"就判AI。要看专业信息的**质感**：
    - **真人专业**（不判AI）：① 引用精确可核验（具体到作者/年份/书名，结论用得准，如 N7 的 Baumeister&Masicampo 2011）；② 实操精确到参数且互相印证（如 N8 的 35℃控水/花多多2号1:1500）；③ 自曝失败/翻车的真实经历（N8 养坏乳突球）；④ 针对具体对象（这盆球、这张图）的细致解读
    - **AI伪专业**（判AI）：① 搬运公开理论当背书但无精确出处；② "放之四海皆准"的通用话术（P13 的"印刷清晰/烟丝油润"）；③ 完美无瑕的人设/案例（P8 小Sun）；④ 可替换性极高（换个商品/主角模板照用，P13）
    - 一句话：**判专业内容看"信息能不能查、有没有手上有泥的颗粒度、人设是否完美得不真实"，而不是看堆了多少专业词**
